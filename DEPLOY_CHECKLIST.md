# デプロイ前チェックリスト

## ✅ 必須項目

- [ ] ビルドが成功する（`npm run build`）
- [ ] ローカル環境で動作確認済み
- [ ] 環境変数がすべて設定されている
- [ ] データベース接続が正常に動作する
- [ ] 初期ユーザーが作成されている

## 🔒 セキュリティ

- [ ] 本番環境用の強力な `NEXTAUTH_SECRET` を生成
- [ ] 本番環境用の `ENCRYPTION_KEY` を生成
- [ ] 初期ユーザーのパスワードを変更
- [ ] `.env` ファイルがGitにコミットされていない

## 📦 デプロイ準備

- [ ] GitHubリポジトリにプッシュ済み
- [ ] デプロイ先のサービス（Vercel/Railway等）のアカウント作成済み
- [ ] 本番環境のPostgreSQLデータベースが準備済み

## 🚀 デプロイ後

- [ ] 本番環境のURLでアクセスできる
- [ ] ログインが正常に動作する
- [ ] 商品管理画面が表示される
- [ ] Shopify連携設定ができる
- [ ] Shopifyから商品情報をインポートできる
